# 查看网络信息
```
# windows 
ipconfig /all

tracert <ip> 

# dns 查询， 使用的哪个dns服务器，查到的地址是多少
nslookup <domain>

# 指定dns服务器进行查询
nslookup   www.google.com  10.238.0.100
向dns服务器10.238.0.100查询google的ip
```
